ROLLING STRADDLE - Complete Autonomous Intraday Trading System
Works with Fyers API
On Scheduling on the server it can start by automatically logging in, placing orders, monitor Positions and Exit by EOd
It is based on the rolling ATM Short straddles of NIFTY/BANKNIFTY options based on the volatility in the market.
Setting can be done in the config.py for instruments, lot size, timeframe etc.
